# Numpy_Data_explorer
# 🌦️ Weather Data Explorer using NumPy

## 📌 Project Overview

Weather Data Explorer is a Python-based data analytics project developed using **NumPy** to demonstrate fundamental and advanced array operations through a weather dataset.

The project showcases how NumPy can be leveraged for efficient data processing, statistical analysis, and weather trend exploration using vectorized computations. It was developed as part of a Data Science Internship to apply NumPy concepts in solving practical analytical problems.

---

## 🎯 Objectives

* Demonstrate core NumPy operations on structured weather data.
* Perform statistical analysis on weather parameters.
* Extract meaningful insights from temperature, humidity, and rainfall data.
* Practice data exploration techniques without relying on higher-level libraries such as Pandas.

---

## 🛠️ Technologies Used

* Python 3
* NumPy
* Pandas

---

## 📊 Dataset Features

The dataset contains weather-related information including:

* City Names
* Month
* Temperature (°C)
* Humidity (%)
* Rainfall (mm)

## 🔍 Analytics Performed

### 1. Monthly Temperature Trends

Analyzed temperature variations across different months to identify seasonal patterns.

### 2. Humidity Analysis

Examined humidity levels across cities and time periods.

### 3. Weather Variability Measurement

Calculated temperature fluctuations to determine weather stability and variability.

### 4. Rainfall Intensity Ranking

Ranked cities and months based on rainfall intensity.

### 5. Anomaly Detection

Identified unusual temperature observations that significantly deviated from normal patterns.

### 6. Temperature–Humidity Correlation Analysis

Measured the relationship between temperature and humidity using correlation techniques.

### 7. Extreme Weather Identification

Detected highest and lowest temperature records within the dataset.
